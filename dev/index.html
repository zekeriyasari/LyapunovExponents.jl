LyapunovExponents.lyapunovs — Method
lyapunovs(ds; kwargs...)

Computes lyapunov exponents of ds.kwargs may include

  • nsteps::Int = 3e4 Number of steps to calculate lypaunov exponents
  • ntrsteps::Int = 0 Number of transient steps for transient steps. Transient steps are taken before calculation of lypaunov exponents
  • dt::Real = 0.01 Step size of integrator.

LyapunovExponents.msf — Method
msf(ds, λ, P; kwargs...)

Master stability function (msf). Returns maximum lyapunov exponent. kwargs may include

  • nsteps::Int = 3e4 Number of steps to calculate lypaunov exponents
  • ntrsteps::Int = 0 Number of transient steps for transient steps. Transient steps are taken before calculation of lypaunov exponents
  • dt::Real = 0.01 Step size of integrator.
